# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.2.0] - 2026-02-11

### Added
- Versioned public APIs and output schema documentation.
- Physics plugin registry with a toy Hall term and plugin template.
- Latent ODE ablations and baseline experiment.
- Paper pipeline with manifest, hashes, and run config bundle.
- Additional validation tests and CI hardening.

### Changed
- Inverse design configuration unified under `mhx.config`.
- Legacy scripts moved to `scripts/legacy` with deprecation shims.
